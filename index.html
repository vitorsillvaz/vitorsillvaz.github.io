HTML 5. Estrutura.

O HTML (HyperText Markup Language) é a linguagem de marcação responsável por estruturar o conteúdo das páginas da web. Ele define elementos como textos, imagens, vídeos, links, tabelas e formulários, organizando a disposição das informações em um site. Além disso, o HTML permite a criação de links para navegação entre páginas, a inserção de mídias como imagens e vídeos, além da construção de formulários para coleta de dados.





CSS. Estilização.

CSS é chamado de linguagem Cascading Style Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site. Pense na decoração da sua página.





JavaScript. Dinamismo.

O JavaScript surgiu como uma tecnologia do lado do navegador para tornar as aplicações Web mais dinâmicas. Ao usar JavaScript, os navegadores passaram a ser capazes de responder a interações do usuário e alterar o layout do conteúdo na página.
